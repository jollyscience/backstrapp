Installation

  1. Get The Code

    Click to download the code from gitHub, or run the following in your terminal from your dev folder.

    $ git clone git://github.com/jollyscience/backstrapp.git backstrapp

  2. Run the Site

    BackStrapp uses jQuery's getJSON() and get() methods, which require a webserver to run. If you're on a mac, we highly recommend MAMP PRO.

    If you plan to use the r.js compiler, point your local server instance to: 

    your_backstrapp_dev_dir/_build

    If you DON'T plan to use the r.js compiler, point your local server instance to: 

    your_backstrapp_dev_dir/_src

    If everything is installed and configured, then you should be able to view your app by navigating to: 

    http://your_local_backstrapp_instance/

  3. Configure r.js as Git Hook (optional)

    Running r.js requires nodeJS to be configured on your local machine.

    Configure your app by editing the file at: 

    your_backstrapp_dev_dir/_src/js/app.build.json

    For more details checkout r.js on github.
