About Backstrapp How it all came to be

Backstrapp was developed with apps in mind. Jolly Science has built and deployed 2 sites using this framework, and are currently developing several more. The framework has served as an incredible boost to productivity, collaboration, and capability. We think you'll like it too!

Features

Global Event Tracking
Using an 'outside-in' philosophy, all classes communicate with a global event model. (Thanks to Derick Bailey at LosTechies for the tip!)
Global Data Model
With the future feature of an external database in mind (MongoDB or CouchDB), we designed Backstrapp with a global data model, which updates dynamically and is accessible to all Classes. This, combined with the global event model, makes for clean, easy-to-trace code, and allows for complex, asynchronous interactions.
JSON Data Structure
We use an external JSON file to control the default pages of the site, and populate a nav bar. It's simple, intuitive, and ideal for quick projects which still need small amounts of dynamic content.
Dynamic Content
Though static content could be put inline to aid in non-obtrusiveness for the site, Backstrapp ships with the default setting of loading all pages dynamically, based on the url.
Easy-to-Manage HTML Content
To make things really easy for small teams of front end developers, Backstrapp's architecture allows for all content pages to be flat HTML files, only containing the body content to be used. These mush be well-formed, but don't have to validate, since they're just being loaded into a container div.
Intuitive Debugging
We like Console. If for no other reason, but to keep Alert() boxes out of our face. So we leveraged Ben Alman's debug script to handle compatibility issues (aghem, I.E.).
Plugin-Ready Architecture
The architecture used allows for a page type of 'app', which connects a url route to a 'module' in Backstrapp. To see this in action, take a look at the 'Test' module that comes with this build.
Page Load Animations
All pages loop through AppView.render(), which uses a simple jQuery animation method to change out page content. It's easy to edit, and fun to try!
Continuous Integration (sorta)
Backstrapp comes equipped with a pre-commit git hook, which runs the r.js compiler. This makes for quick and easy deployment of a single JS file, while maintaining a clean working environment with multiple MVC-ish files for your app.
