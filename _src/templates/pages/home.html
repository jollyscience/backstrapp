Backstrapp A faster way to write better apps.

Backstrapp is boilerplate for app development that is easy-to-use, powerful, and well-documented. We've compiled our favorite tools into one collection of files to act as a boilerplate for future web apps.

Git the Code!

Inspiration

This is an attempt to build a bootstrap for Backbonejs, requirejs, AMD, and r.js, leveraging the architecture provided by Thomas Davis at backbonetutorials.com

Dependencies

jQuery
a fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, and Ajax interactions for rapid web development.
UnderscoreJS
a utility-belt library for JavaScript that provides a lot of the functional programming support that you would expect in Prototype.js (or Ruby), but without extending any of the built-in JavaScript objects.
BackboneJS
supplies structure to JavaScript-heavy applications by providing models with key-value binding and custom events, collections with a rich API of enumerable functions, views with declarative event handling, and connects it all to your existing application over a RESTful JSON interface.
HandlebarsJS
provides the power necessary to let you build semantic templates effectively with no frustration.
RequireJS
a JavaScript file and module loader. It is optimized for in-browser use, but it can be used in other JavaScript environments, like Rhino and Node.
r.js
Runs RequireJS in Node and Rhino, and used to run the RequireJS optimizer
Bootstrap
a toolkit from Twitter designed to kickstart development of webapps and sites. It includes base CSS and HTML for typography, forms, buttons, tables, grids, navigation, and more.
